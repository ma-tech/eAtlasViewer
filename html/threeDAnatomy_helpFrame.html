Keyboard Controls

  • pan (side to side, up and down)—press d and drag mouse with left button pressed
  • zoom (in and out)—press s and drag mouse (vertically) with left button pressed
  • reduce opacity—press <
  • increase opacity—press >
  • set Camera position—press C
  • set Home position—press H
  • go to Home position—press h
  • render as smooth surface—press s
  • render as wire frame—press w
Note


The above controls will only function if the 3d window has 'focus'. You can give the 3d window focus by clicking inside it.

Warning


Displaying too many 3d surfaces in a web browser can cause your computer to become slow or un-responsive.

If this happens you may wish to limit the number of anatomy components which are selected for display (checked in the tree).

(Closing and re-starting your web-browser should restore normal browser performance.)
